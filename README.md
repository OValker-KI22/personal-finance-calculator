# Personal Finance Calculator

Калькулятор для учёта личных финансов с автоматическим CI/CD pipeline.

## Функциональность

- Учёт доходов и расходов
- Категоризация расходов
- Расчёт баланса
- Экспорт в CSV
- Автоматическое тестирование
- Проверка код-стайла

## Установка

```bash
git clone https://github.com/ваш-username/personal-finance-calculator.git
cd personal-finance-calculator
pip install -r requirements.txt
